Usage

Before building, make sure you create a .env file in the root. To the root, add your firebase API key as follows:

VITE_FIREBASE_API_KEY=...
VITE_BACKEND_URL=...

To build the project, run:

vite build

or

npm run build

To start the development server, run:

vite preview

or

npm run dev
